## Vesting Formula

### For Each Batch of Auto-Staked

```solidity
Vesting Per Year = BatchAmount / 15

For Year 1 batch (94,000 IONX):
- Year 1: 94,000 / 15 = 6,267 vested
- Year 2: 94,000 / 15 = 6,267 vested
- ...
- Year 15: Remaining amount vested

For Year 2 batch (47,000 IONX):
- Year 2: 47,000 / 15 = 3,133 vested
- Year 3: 47,000 / 15 = 3,133 vested
- ...
- Year 16: Would vest but emission stopped
```

### Linear Vesting Calculation

```javascript
function calculateVested(batchAmount, yearsElapsed) {
  const vestingPeriod = 15;
  const vestedAmount = (batchAmount * yearsElapsed) / vestingPeriod;
  
  if (yearsElapsed >= vestingPeriod) {
    return batchAmount; // Fully vested
  }
  
  return vestedAmount; // Partial vesting
}
```
